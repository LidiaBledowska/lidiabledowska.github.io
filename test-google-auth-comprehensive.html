🛠️ Możliwe rozwiązania

⚠️ Problem z domeną

Jeśli błąd to "unauthorized-domain":

  1. Idź do Firebase Console
  2. Wybierz projekt aplikacje-3068f
  3. Authentication → Settings → Authorized domains
  4. Dodaj: localhost, localhost:8081

🚫 Popup blokowany

Jeśli popup jest blokowany:

  1. Sprawdź ikonę popup w pasku adresu
  2. Dodaj localhost do wyjątków
  3. Wyłącz blokadę popup dla tej strony

🔒 Problem z HTTPS

Google Auth może wymagać HTTPS:

  1. Spróbuj na https://localhost:8081
  2. Lub użyj ngrok do tunelowania
  3. Sprawdź czy błąd znika na HTTPS

🔑 Konfiguracja API

Sprawdź konfigurację Firebase:

  1. Czy projekt jest aktywny?
  2. Czy Google Auth jest włączony?
  3. Czy API key jest poprawny?
