🛠️ Rozwiązywanie problemów

Jeśli nie działa:

  1. Sprawdź popup: Czy przeglądarka nie blokuje popup?
  2. Sprawdź domenę: Czy localhost:8082 jest w Firebase Console?
  3. Sprawdź Google Auth: Czy jest włączony w Firebase?
  4. Sprawdź błędy: Otwórz Developer Tools (F12) → Console

Najczęstsze błędy:

  • auth/unauthorized-domain → Dodaj domenę w Firebase Console
  • auth/popup-blocked → Odblokuj popup w przeglądarce
  • auth/popup-closed-by-user → Nie zamykaj popup
