🔍 Debug logowania Google

Ta strona pomoże zdiagnozować problem z logowaniem przez Google.

Kliknij przycisk testowy aby rozpocząć diagnozę...

🎯 Możliwe przyczyny problemów:

  • Konfiguracja Firebase: Nieprawidłowy API key lub projekt
  • Domeny autoryzowane: localhost może nie być autoryzowany w Firebase Console
  • Popup blokowany: Przeglądarka może blokować popup logowania
  • Wersja Firebase SDK: Niezgodność wersji modułów
  • HTTPS wymagane: Google Auth może wymagać HTTPS

📋 Instrukcje manualnego testu:

  1. Otwórz stronę logowania
  2. Otwórz Developer Tools (F12) i przejdź do zakładki Console
  3. Kliknij przycisk "Zaloguj się przez Google"
  4. Sprawdź czy pojawiają się błędy w konsoli
  5. Sprawdź czy otwiera się popup logowania Google
